# Recipe Book App

A simple Flutter application to manage and view recipes.

## Features
- **Recipe List**: View a collection of delicious recipes with images.
- **Recipe Details**: See ingredients and step-by-step instructions for each recipe.
- **Search**: Quickly find recipes by title.
- **Basic Data Storage**: Recipes are managed within the app's data structures.

## Requirements Met
1. **Flutter Widgets**: Utilizes `ListView`, `Card`, `TextField`, `Image.network`, and more.
2. **UI Design**: Includes a list view, detail view, and search bar.
3. **Data Storage**: Uses a structured `Recipe` model and an in-memory list (can be extended to persistent storage).

## How to Run
1. Ensure you have Flutter installed.
2. Run `flutter run` in the terminal.

## Deliverables
- `lib/main.dart`: Contains the full application code.
- To generate the APK, run: `flutter build apk --release`.
